# 任务管理
基本地址`task`

## /task/task_set_detail
Desc: 设置任务的细节信息,仅仅添加,即在不存在的情况下有效。不允许对已存在的进行修改

Method: POST

Param:
* task_id (require) 任务ID
* detail (require) 细节信息

#### 错误状态，错误标记`190`
* `19000` 非教师用户，无权限
* `19001` 传入的任务ID有误
* `19002` 传入的详细信息不能为空
* `19003` 任务未找到
* `19004` 该任务无权修改
* `19005` 该任务细节不为空，无法修改
* `19006` 无法更新任务细节
